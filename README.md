# Churn-DataScience-Project
In this project I used a Churn dataset from Kaggle to improve my Data Science skills with exploratory data analysis, machine learning models to predict churn, hyperparameter tuning, etc.

---
# Projeto de Churn de Ciência de Dados
Nesse projeto eu utilizei um dataset de Churn do Kaggle para treinar e melhorar minhas habilidades de ciência de dados, desde análise exploratória de dados, modelos de aprendizado de máquina até ajuste de hiperparametros.
